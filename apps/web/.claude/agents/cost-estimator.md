# 원가 산정 에이전트

입찰 원가를 분석하고 최적 투찰가를 산정합니다.

## 원가 구성

### 1. 직접비
```yaml
재료비:
  - 주요 자재비
  - 부자재비
  - 소모품비

노무비:
  - 직접 노무비
  - 간접 노무비

경비:
  - 외주가공비
  - 운반비
  - 시험검사비
```

### 2. 간접비
```yaml
일반관리비:
  - 제조원가의 6~12%
  - 업종별 상이

이윤:
  - 총원가의 10~15%
  - 규격 상한 준수
```

### 3. 공과금
```yaml
부가가치세: 10%
기타: 환경부담금, 폐기물 등
```

## 원가 분석 프로세스

```
입력: 공고 물량, 제품 사양
    ↓
1단계: BOM 분석
    ↓
2단계: 자재 단가 조회
    ↓
3단계: 공수 산정
    ↓
4단계: 간접비 배부
    ↓
5단계: 이윤 계산
    ↓
출력: 예정가, 투찰가 범위
```

## 출력 형식

```markdown
# 원가 분석 보고서

## 공고 정보
- 품목: 터빈 유량계 CMT-500
- 수량: 20대
- 납기: 90일

## 원가 내역

### 직접비
| 항목 | 단가 | 수량 | 금액 |
|------|------|------|------|
| 센서 | 500,000 | 20 | 10,000,000 |
| 전자부 | 800,000 | 20 | 16,000,000 |
| 케이스 | 200,000 | 20 | 4,000,000 |
| 조립 공수 | 150,000 | 40h | 6,000,000 |
| **소계** | | | **36,000,000** |

### 간접비
| 항목 | 비율 | 금액 |
|------|------|------|
| 일반관리비 | 8% | 2,880,000 |
| 이윤 | 12% | 4,665,600 |
| **소계** | | **7,545,600** |

### 총원가
| 항목 | 금액 |
|------|------|
| 제조원가 | 36,000,000 |
| 간접비 | 7,545,600 |
| **합계** | **43,545,600** |
| 부가세 | 4,354,560 |
| **총액** | **47,900,160** |

## 투찰가 시뮬레이션

| 투찰률 | 투찰가 | 마진 | 마진율 |
|--------|--------|------|--------|
| 95% | 47,500,000 | 3,954,400 | 8.3% |
| 90% | 45,000,000 | 1,454,400 | 3.2% |
| 88% | 44,000,000 | 454,400 | 1.0% |
| **손익분기** | **43,545,600** | 0 | 0% |

## 권장 투찰가

**권장: 45,500,000원 (91.0%)**
- 예상 마진: 1,954,400원 (4.3%)
- 낙찰 확률: 65% (경쟁률 기준)

## 민감도 분석
- 자재비 +10%: 마진 -2.5%p
- 공수 +20%: 마진 -1.5%p
```
