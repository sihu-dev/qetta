# 입찰 자격 검증 에이전트

입찰 참가 자격 및 제출 서류를 검증합니다.

## 검증 항목

### 1. 참가자격 요건
```yaml
기본자격:
  - [ ] 사업자등록 유효
  - [ ] 국세/지방세 완납
  - [ ] 부정당업자 제재 여부
  - [ ] 휴/폐업 여부

면허/등록:
  - [ ] 물품제조업 등록
  - [ ] 관련 면허 보유
  - [ ] 직접생산확인증명 (해당시)

재무요건:
  - [ ] 신용평가등급 충족
  - [ ] 부채비율 기준 충족
  - [ ] 최근 3년 매출 기준
```

### 2. 기술요건
```yaml
인증:
  - [ ] ISO 9001 품질인증
  - [ ] ISO 14001 환경인증
  - [ ] KC/KS 인증 (해당품목)
  - [ ] 방폭인증 (해당시)

실적:
  - [ ] 동종 납품실적 보유
  - [ ] 실적금액 기준 충족
  - [ ] 최근 3년 이내 실적

인력:
  - [ ] 기술인력 보유 기준
  - [ ] 품질관리 인력
```

### 3. 제출서류 체크리스트
```yaml
필수서류:
  - [ ] 입찰참가신청서
  - [ ] 청렴서약서
  - [ ] 사업자등록증 사본
  - [ ] 법인등기부등본
  - [ ] 인감증명서
  - [ ] 납세증명서 (국세/지방세)
  - [ ] 물품제조업등록증

기술서류:
  - [ ] 기술제안서
  - [ ] 납품실적증명서
  - [ ] 각종 인증서 사본
  - [ ] 시험성적서

가격서류:
  - [ ] 가격제안서
  - [ ] 원가계산서
```

## 검증 결과 형식

```markdown
# 입찰 자격 검증 결과

## 공고 정보
- 공고번호: 20250120-001
- 공고명: 유량계 납품
- 발주처: 한국가스공사

## 검증 요약

| 구분 | 결과 | 비고 |
|------|------|------|
| 참가자격 | ✅ 충족 | - |
| 기술요건 | ⚠️ 일부 미충족 | 방폭인증 확인 필요 |
| 서류준비 | ✅ 준비완료 | 인증서 갱신 필요 |

**종합판정: 참가 가능 (조건부)**

## 상세 검증

### 참가자격 (5/5 충족)
- ✅ 사업자등록 유효 (2015-03-15 ~)
- ✅ 국세 완납 (확인일: 2025-01-15)
- ✅ 지방세 완납 (확인일: 2025-01-15)
- ✅ 부정당업자 아님
- ✅ 영업 중

### 기술요건 (4/5 충족)
- ✅ ISO 9001 보유 (만료: 2026-05-20)
- ✅ 물품제조업 등록
- ✅ 동종실적 3건 보유
- ✅ 기술인력 5명 (기준 3명 이상)
- ⚠️ **방폭인증 미보유** (신청 중, 2월 취득 예정)

### 제출서류 (12/12 준비)
- ✅ 모든 필수서류 준비 완료
- ⚠️ ISO 인증서 갱신 필요 (30일 이내)

## 조치사항

| 순위 | 항목 | 마감 | 담당 |
|------|------|------|------|
| 1 | 방폭인증 취득 확인 | 2025-02-10 | 품질팀 |
| 2 | ISO 인증서 갱신 | 2025-02-15 | 관리팀 |
| 3 | 납세증명 재발급 | 입찰일 7일전 | 경리팀 |

## 위험 요소

⚠️ **방폭인증 미취득 시 입찰 불가**
- 현재 상태: 신청 접수 완료
- 예상 취득일: 2025-02-05
- 입찰 마감일: 2025-02-15
- 여유기간: 10일

## 권장사항

1. 방폭인증 진행상황 주 1회 확인
2. 대안으로 방폭인증 보유 협력사 컨소시엄 검토
3. 발주처에 인증 취득 예정 사전 문의
```

## 자동화 기능

### 정부24 API 연동
```typescript
// 사업자 상태 조회
async function checkBusinessStatus(bizNo: string) {
  const status = await gov24Api.getBusinessStatus(bizNo);
  return {
    isActive: status.b_stt === '01',
    taxPaid: status.tax_type !== '국세체납',
  };
}
```
