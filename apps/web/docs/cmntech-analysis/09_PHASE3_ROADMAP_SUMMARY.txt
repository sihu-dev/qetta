## 4. 스프레드시트 열 구성

| 열 | 설명 | 데이터 타입 |
|----|------|-------------|
| No | 순번 | number |
| 공고명 | 입찰 제목 | text |
| 발주기관 | 기관명 | text |
| 추정가격 | 금액 (₩) | numeric |
| 마감일 | 날짜 | date |
| 상태 | 진행 상태 | dropdown |
| 우선순위 | high/medium/low | dropdown |
| 매칭점수 | 제품 매칭 % | numeric |
| AI 요약 | AI 분석 결과 | text |

---

## 5. AI 셀 함수

지원 함수:
- =AI(prompt) - 자유 프롬프트
- =AI_SUMMARY(range) - 범위 요약
- =AI_SCORE(bidId) - 낙찰 확률
- =AI_MATCH(bidId) - 제품 매칭
- =AI_EXTRACT(url, field) - PDF 추출
