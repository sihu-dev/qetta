# BIDFLOW 엔터프라이즈 - AI 바우처 사업 기획

> **지원금**: 정보통신산업진흥원 AI 바우처 (최대 3억원)
> **지원 비율**: 정부 70% + 자부담 30%

---

## 1. 핵심 기능

### 1.1 경쟁사 모니터링

기능:
- 실시간 경쟁사 입찰 추적
- 낙찰 패턴 분석
- 가격 전략 인사이트

데이터 소스:
- 나라장터 낙찰 공개 데이터
- 조달청 API
- 공공데이터포털

### 1.2 스마트 매칭

기능:
- 제품-공고 자동 매칭
- 적합성 점수 예측 (0-100)
- 추천 우선순위

매칭 알고리즘:
- 키워드 매칭 (40%)
- 규격 호환성 (30%)
- 가격 범위 (20%)
- 과거 낙찰 이력 (10%)

### 1.3 자동 제안서

기능:
- 제안서 초안 생성
- 템플릿 학습
- 섹션별 커스터마이징

생성 섹션:
- 요약 (Executive Summary) - Claude
- 기술 제안서 - Claude
- 가격 제안서 - Gemini
- 이행 일정 - Gemini
- 회사 소개 - Template

### 1.4 시장 인사이트

기능:
- 산업별 트렌드 분석
- 예산 사이클 예측
- 신규 기회 발굴

## 4. 기대 효과

### 정량적 효과

| 지표 | 현재 | 도입 후 | 개선율 |
|------|------|---------|--------|
| 입찰 분석 시간 | 4시간/건 | 30분/건 | 87% 감소 |
| 제안서 작성 | 3일/건 | 4시간/건 | 83% 감소 |
| 적합 공고 발굴 | 10건/주 | 50건/주 | 400% 증가 |
| 낙찰률 | 15% | 25% | 67% 향상 |

### 정성적 효과

- 경쟁 전략 고도화
- 데이터 기반 의사결정
- 영업 인력 업무 효율화
- 신규 시장 진출 기회 발굴
