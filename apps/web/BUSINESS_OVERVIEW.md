## Executive Summary

### 한 줄 요약
**제조업 SME를 위한 AI 기반 입찰 자동화 SaaS - 입찰 준비 시간 70% 단축**

### 핵심 지표 (v0.1.0 기준)

```
✓ 92% - 평균 제품 매칭 정확도
✓ 5개 - 연동 제품 (CMNTech 유량계/열량계)
✓ 150+ - 월간 분석 공고 수 (예상)
✓ 3.2x - 입찰 참여율 증가 (목표)
✓ 45+ - 데이터 소스 (나라장터, TED, K-water 등)
```

### 타겟 고객
- **1차**: (주)씨엠엔텍 (유량계/열량계 제조사)
- **2차**: 중소 제조업 입찰 담당자
- **3차**: 공공조달 컨설턴트

### 수익 모델
- **Starter**: 월 \99,000 (3개 제품, 기본 AI 함수)
- **Pro**: 월 \299,000 (무제한 제품, 전체 AI 함수)
- **Enterprise**: 맞춤형 계약 (\5M+/년, AI 바우처 지원)

## 문제 정의

### 현재 제조업 입찰의 문제점

#### 1. 정보 과부하 (Information Overload)

```
문제: 하루 평균 500+ 입찰 공고 발행 (나라장터만)
결과: 담당자가 관련 공고를 찾기 어려움
비용: 하루 2-3시간 소요, 월 60+ 시간 낭비
```

#### 2. 수동 매칭의 한계

```
문제: 제품-공고 매칭을 수작업으로 수행
결과:
  - 놓친 기회: 월 평균 15-20건
  - 오판 비율: 30-40% (시간 낭비)
비용: 연간 5,000만원+ 기회 비용 손실
```

#### 3. 분산된 데이터 소스

```
문제: 45+ 입찰 플랫폼에 데이터 분산
  - 나라장터 (G2B)
  - TED (EU 공공조달)
  - K-water, 한전, 각 지자체 별도 시스템
결과: 통합 모니터링 불가능
비용: 플랫폼당 하루 30분 = 총 20시간/일
```

#### 4. 제안서 작성 부담

```
문제: 공고당 제안서 작성에 평균 8-12시간 소요
결과: 월 최대 10-15건만 입찰 참여 가능
비용: 우수 공고 선별 실패 → 낙찰률 하락
```

### 정량적 영향

| 항목 | 현재 상황 | 연간 비용 |
|------|-----------|----------|
| 공고 모니터링 시간 | 월 60시간 | \25M (인건비) |

## 솔루션

### BIDFLOW의 핵심 가치

#### 1. 자동 공고 수집 (Automated Crawling)

```
Before BIDFLOW:
  - 45개 플랫폼 수동 확인
  - 하루 2-3시간 소요

After BIDFLOW:
  - 자동 수집 (매일 2회)
  - 0시간 소요 → 100% 시간 절감
```

#### 2. AI 제품 매칭 (Enhanced Matcher)

```
Before BIDFLOW:
  - 수동 키워드 검색
  - 매칭 정확도: 60-70%

After BIDFLOW:
  - AI 자동 매칭 (3단계 알고리즘)
  - 매칭 정확도: 92%
  - 놓친 기회: 0건
```

**3단계 매칭 알고리즘**:
1. **키워드 매칭** (100점): 강한 키워드 + 약한 키워드 - 제외 키워드
2. **규격 매칭** (25점): 파이프 DN 사이즈 자동 추출 및 비교
3. **기관 매칭** (50점): 45개 주요 발주기관 정규화 및 가중치

#### 3. AI 스마트 함수 (5종)

| 함수 | 기능 | 가치 |
|------|------|------|
| **=AI_SUMMARY()** | 공고 2-3문장 요약 | 읽기 시간 90% 단축 |
| **=AI_SCORE()** | 낙찰 가능성 0-100% | 우선순위 명확화 |
| **=AI_MATCH()** | 최적 제품 추천 | 오판 방지 |
| **=AI_KEYWORDS()** | 핵심 키워드 3개 | 빠른 스캔 |
| **=AI_DEADLINE()** | 마감일 분석 + 일정 제안 | 미스 방지 |

#### 4. 맞춤형 제안서 생성 (AI Generator)

```
Before BIDFLOW:
  - 공고당 8-12시간 소요
  - 월 최대 10-15건 작성

After BIDFLOW:
  - AI 자동 생성 (1-2시간)
  - 월 30-50건 작성 가능
  - 3.2x 입찰 참여율 증가
```

### 사용자 워크플로우

```
1. 아침 출근
   → BIDFLOW 대시보드 접속
   → "새로운 고신뢰도 공고 3건" 알림

2. 공고 검토 (5분)
   → AI_SCORE() 92% 공고 우선 확인
   → AI_SUMMARY() 2줄 요약 읽기
   → AI_MATCH() 추천 제품 확인

3. 의사결정 (3분)
   → "BID" 버튼 클릭
   → AI 제안서 생성 요청

4. 제안서 검토 (1시간)
   → AI 생성 제안서 다운로드
   → 회사 정보 커스터마이징
   → 제출

총 소요 시간: 1.1시간 (기존 8-12시간 → 91% 단축)

## 시장 분석

### 시장 규모

#### 국내 공공조달 시장

```
총 규모: 연간 160조원 (2024년 기준)
  - 중앙정부: 70조원
  - 지방정부: 60조원
  - 공기업: 30조원

제조업 비중: 40% (64조원)
```

#### TAM (Total Addressable Market)

```
국내 제조업체 수: 약 50만개
  - 중소기업 (직원 50-300명): 5만개
  - 입찰 참여 제조업: 약 2만개

TAM = 2만개 × \300,000/월 = 72억원/월 = 864억원/년
```

#### SAM (Serviceable Available Market)

```
초기 타겟: 유량계/열량계/계측기 제조업
  - 국내 업체 수: 약 200개
  - 활발히 입찰 참여: 50개

SAM = 50개 × \300,000/월 = 1,500만원/월 = 1.8억원/년
```

#### SOM (Serviceable Obtainable Market)

```
3년 목표 (2028년):
  - 고객사 수: 20개
  - ARPU: \400,000/월 (평균)

SOM = 20개 × \400,000/월 = 800만원/월 = 9,600만원/년
```

### 시장 트렌드

#### 1. 공공조달 디지털 전환 가속

```
정부 정책:
  - 나라장터 2.0 (2025년)
  - 스마트 조달 시스템 구축

기회: BIDFLOW는 이미 AI 기반 → 정책 방향과 일치
```

#### 2. 중소기업 AI 바우처 지원 확대

```
2025년 AI 바우처 예산: 500억원
  - 기업당 최대 7,000만원 지원
  - 제조업 우선 지원

기회: Enterprise 플랜 AI 바우처로 진입장벽 제거
```

#### 3. 글로벌 조달 시장 확대

```

  - 연간 2조 유로 (약 2,800조원)

## 경쟁 우위

### 1. 제품 특화 (Product-Specific Matching)

```
차별점: 범용 키워드 검색이 아닌, 제품 카탈로그 기반 매칭
경쟁사: 단순 키워드 검색
BIDFLOW: 5개 제품 × 각 10+ 키워드 × 규격 정보 = 정밀 매칭
```

### 2. 3단계 매칭 알고리즘 (특허 출원 예정)

```
독창성:
  - 키워드 (100점) + 규격 (25점) + 기관 (50점) = 175점 체계
  - 파이프 DN 사이즈 자동 추출 (10가지 패턴)
  - 45개 발주기관 정규화 및 가중치

경쟁사: 이런 세밀한 알고리즘 없음
```

### 3. 도메인 전문성 (Domain Expertise)

```
강점: 씨엠엔텍과 25년 유량계 업력 기반 협업
  - 제품 스펙 정확히 이해
  - 발주기관별 선호도 파악
  - 입찰 실패 케이스 학습

경쟁사: 범용 솔루션, 도메인 지식 부족
```

### 4. 글로벌 데이터 (TED, SAM.gov)

```
차별점: 국내 + 해외 입찰 통합 모니터링
  - TED API 연동 (EU 2,800조원 시장)
  - SAM.gov 크롤링 (미국)

경쟁사: 국내 나라장터만 지원
```

### 5. AI 스마트 함수 (Excel-like UX)

```
혁신: Excel 수식처럼 쉬운 AI 활용
  - =AI_SUMMARY(A2) → 즉시 요약
  - 학습 곡선 없음

경쟁사: 복잡한 AI 설정 필요
```
