│                                                                         │

│  ┌──────────────────┬──────────────────────────────────┐               │
│  │ 전류 (mA)        │ 의미                              │               │
│  ├──────────────────┼──────────────────────────────────┤               │
│  │ 4.0              │ 0% (최소 유량)                    │               │

│                                                                         │

│  │ 센서             │ 측정값                            │               │
│  ├──────────────────┼──────────────────────────────────┤               │
│  │ 수위 (mm)        │ 침전물 위에서 액체 표면까지       │               │
│  │ 유속 (m/s)       │ 액체 평균 유속                    │               │
