## 1. Executive Summary

### 1.1 원라인 피치
```
"제조업 SME를 위한 AI 기반 글로벌 입찰 자동화 플랫폼"
```

### 1.2 핵심 문제
- 제조업 SME는 해외 공공입찰 정보 접근이 어려움
- 입찰 공고 분석에 과도한 시간 소요 (월 80시간+)
- 영어/EU 언어 장벽으로 기회 손실
- 적합한 입찰 선별 노하우 부재

### 1.3 솔루션
```
┌─────────────────────────────────────────────────────────────────┐
│  Qetta = AI가 입찰 공고를 수집 → 분석 → 매칭 → 알림          │
│                                                                 │
│  • 45+ 글로벌 조달 플랫폼 통합                                   │
│  • Claude AI 기반 스마트 매칭                                    │
│  • 스프레드시트 UX (익숙한 인터페이스)                           │
│  • 한국어 번역 + 요약 자동화                                     │
└─────────────────────────────────────────────────────────────────┘
```

### 1.4 비즈니스 모델
```
Primary: AI 바우처 기업 컨설팅 (7,000만원/건)
Secondary: B2B SaaS 구독 (월 50-200만원)
Tertiary: 입찰 성공 수수료 (낙찰가 1-3%)
```

### 1.5 핵심 지표 (Unit Economics)
| 지표 | 값 | 설명 |
|------|-----|------|
| CAC | 150만원 | AI 바우처로 절감 |
| LTV | 3,600만원 | 3년 유지, 연 1,200만원 |
| LTV/CAC | **24.0** | 건강한 비율 |
| Payback | 1.5개월 | 빠른 회수 |
