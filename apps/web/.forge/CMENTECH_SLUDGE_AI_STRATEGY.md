# 씨엠엔텍 슬러지 AI 사업 통합 전략

**작성일**: 2025년 12월 21일
**버전**: 1.0
**목적**: bidflow 플랫폼 기반 슬러지/수처리 AI SaaS 사업 확장

---

## Executive Summary

씨엠엔텍(CMENTECH)은 25년간 1.5만대 유량계를 설치한 **국내 최고 계측기 기업**입니다. 2026년 **바이오가스 생산목표제** 시행으로 슬러지 처리 시설의 디지털 전환이 강제되는 시점에, 씨엠엔텍의 **유량계 하드웨어**와 **WATROPUR 슬러지 장비**, 그리고 **bidflow SaaS 플랫폼**을 결합하면 **블루오션 시장을 선점**할 수 있습니다.

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   씨엠엔텍 유량계  +  WATROPUR 슬러지 장비  +  SludgeAI Pro SaaS       │
│   ─────────────────────────────────────────────────────────────────     │
│                                                                         │
│   = 하드웨어-소프트웨어 통합 슬러지 관리 플랫폼 (국내 유일)            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 데이터베이스 스키마 (신규)

```sql
-- 슬러지 처리 시설
CREATE TABLE sludge_sites (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(200) NOT NULL,
    type VARCHAR(50), -- 'public_wwtp', 'private_wwtp', 'biogas'
    address TEXT,
    capacity_m3_day DECIMAL,
    organization_id UUID REFERENCES organizations(id),
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 센서 (씨엠엔텍 유량계 연동)
CREATE TABLE sludge_sensors (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    site_id UUID REFERENCES sludge_sites(id),
    name VARCHAR(200) NOT NULL,
    type VARCHAR(50), -- 'flow_ur1010', 'flow_sl3000', 'temp', 'ph'
    model VARCHAR(100), -- 'UR-1010PLUS', 'SL-3000PLUS'
    protocol VARCHAR(50), -- 'modbus', '4-20mA', 'rs485'
    address INT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 센서 데이터 (TimescaleDB 확장 권장)
CREATE TABLE sludge_readings (
    time TIMESTAMPTZ NOT NULL,
    site_id UUID REFERENCES sludge_sites(id),
    sensor_id UUID REFERENCES sludge_sensors(id),
    value DOUBLE PRECISION NOT NULL,
    unit VARCHAR(20),
    quality SMALLINT DEFAULT 100
);

-- AI 예측 결과
CREATE TABLE sludge_predictions (
    id SERIAL PRIMARY KEY,
    site_id UUID REFERENCES sludge_sites(id),
    prediction_type VARCHAR(50), -- 'sludge_volume', 'biogas', 'failure'
    predicted_at TIMESTAMPTZ DEFAULT NOW(),
    target_date DATE,
    predicted_value DOUBLE PRECISION,
    actual_value DOUBLE PRECISION,
    model_version VARCHAR(50)
);

-- 정책 보고서
CREATE TABLE sludge_reports (
    id VARCHAR(50) PRIMARY KEY,
    site_id UUID REFERENCES sludge_sites(id),
    report_type VARCHAR(50), -- 'monthly_biogas', 'quarterly'
    period_start DATE,
    period_end DATE,
    status VARCHAR(20), -- 'draft', 'submitted', 'approved'
    data JSONB,
    file_url TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 3.3 AI 함수 확장

| 기존 (입찰용) | 신규 (슬러지용) | 구현 |
|---------------|----------------|------|
| `=AI_SUMMARY()` | `=SLUDGE_PREDICT()` | 슬러지 발생량 예측 |
| `=AI_SCORE()` | `=BIOGAS_ESTIMATE()` | 바이오가스 생산량 추정 |
| `=AI_MATCH()` | `=SENSOR_RECOMMEND()` | 최적 센서 추천 |
| `=AI_KEYWORDS()` | `=POLICY_REPORT()` | 정책 보고서 생성 |
| `=AI_DEADLINE()` | `=FAILURE_ALERT()` | 장비 고장 예측 |
