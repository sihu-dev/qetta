## 1. 페르소나 정의

### 김영수 과장 프로필

```yaml
이름: 김영수
직책: 영업기획팀 과장
나이: 42세
근속: 8년 (CMNTech)
주요 업무: 공공 입찰 관리, 제안서 작성, 영업 전략

Pain Points:
  - 매일 3시간+ 수동 검색 (나라장터, 조달청, 지방계약)
  - 여러 사이트 로그인/검색 반복
  - 공고 놓침률 60%+
  - 제안서 매번 처음부터 작성 (3시간+)
  - 경쟁사 동향 파악 어려움
  - 팀장 압박: "왜 이 공고 놓쳤어?"

Goals:
  - 입찰 검색 시간 최소화
  - 공고 놓침 제로
  - 제안서 작성 효율화
  - 낙찰률 향상 (현재 15% → 목표 30%+)
```

## 6. AI 바우처 외주 개발 명세

### 6.1 프로젝트 개요

```yaml
프로젝트명: BIDFLOW AI 입찰 예측 시스템 고도화
총 예산: ₩100,000,000
  - 정부 지원: ₩70,000,000 (70%)
  - BIDFLOW 자부담: ₩30,000,000 (30%)
기간: 6개월 (2025.03 ~ 2025.08)
수행 체계: BIDFLOW (주관) × CMNTech (참여)
```

### 6.2 개발 모듈

| 모듈 | 예산 | 핵심 기능 |
|------|------|-----------|
| 낙찰가 예측 AI | ₩30,000,000 | XGBoost 기반, 정확도 85%+ |
| 제안서 자동 생성 | ₩35,000,000 | GPT-4o, 초안 완성도 70%+ |
| 경쟁사 분석 대시보드 | ₩20,000,000 | 실시간 수집, 점유율 시각화 |
| 기관 선호도 분석 | ₩15,000,000 | 100개+ 기관 프로파일 |

### 6.3 기술 스택

```yaml
Backend: Python 3.11+, FastAPI
ML: XGBoost, LightGBM, Scikit-learn
LLM: OpenAI GPT-4o, LangChain
Frontend: Next.js 14, Recharts
Database: PostgreSQL, Redis
Infra: AWS (EC2, RDS, S3)
```

### 6.4 산출물

1. 학습된 ML 모델 (.pkl)
